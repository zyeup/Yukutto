# Yukuttoの概要 ([https://web.yukutto.com/])

## 利用方法
1.「新しく投稿を作成」をクリックし、投稿のタイトルを決める

2.表示されている地図をクリックすると、クリックした場所にピンが表示される  

3.タイトルを入力して保存を押すと、マーカーを保存することができる
